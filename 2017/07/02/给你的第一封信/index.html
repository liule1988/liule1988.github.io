给你的第一封信

2017-07-02

#给你的信

嗨，真的是好久不见呢。久到我都忘了最后一次和你见面的场景。这些年来，其实我每天都在想，要是当时我不那么着急，不那么幼稚，现在的情况会不会好很多呢？老实说，现在的我越来越害怕哪天你突然联系我，说你要结婚了，我真的不知道收到这样的信息我会怎么样，会崩溃大哭一场，还是会纵酒过度消沉意志，总之，我很难想象我会真心去祝福你，这种感觉心如刀割。

其实，长时间的不相见也有好处，我们彼此都有了很多新朋友，都有各自不知道的精彩的，痛苦的各式各样的故事和经力。我在想，万一哪天，我们真的在一起了，也就有更多的话了，来说说这些年的变化。我的经历还算丰富，相信你的也更让我着迷。

时间过的可真快，转眼已经毕业工作一年了。我想去你的城市看你，但我不知道如何开口，不知道你心里还是否愿意让我继续联系你。上个月，回家了一趟，本想在回北京的那天，带上一束花，风尘仆仆地去看你。哪怕就是和你说上十分钟的话，那么我回北京也就不那么孤独。可是内心的枷锁还是无法打开，加上天公不作美，也就无法如愿。

我为什么会想到以这种方式来和你交流呢？不是我人格分裂，而是我最近重温了「少年张三丰」，很经典，看完我很感动。其中，易继风有一阵子去大漠了，但他经常用书信和张君宝联系，说说他在这里的生活和想法。我觉得很好，现在如果还有书信交流，那是多么美好的事情啊。

明道红死的时候我哭了，真不是我矫情，我也不知道为啥，这部剧整体嘻嘻哈哈的，可是一个活蹦乱跳，一直相信宿命的女子就这么突然走了，我的眼泪也停不下来了。相反，最后，君宝和蓉蓉的分别，我倒没有那么伤感，也许，有一天我真的能做到彻底忘记，也许那天我也就不用在再给你写信了。

对了，我记得你以前说要去看电影「钢铁侠3」，当时，我对漫威电影无感，但是现在每一部我都看了不止一遍，不知何时，能再与你一起，复仇者联盟3？我不知道。我觉得漫威的超级英雄们真的好帅，不知道你最喜欢哪个？我猜应该是钢铁侠吧。我也挺喜欢。

一年时间，我的技术水平有些许提升，但是，离大牛还差的很远，我希望下次再给你写信的时候，可以获得一些提高。最近工作上的问题，已经不能考之前的积累来完成了，这提醒我要开始充电了。我相信，只要我努力，一定可以出色的完成任务的。

对了，我也想过换工作的事，现在的公司不错，但前端开发的大牛还是很少，对我的成长来说这是不利的。但是我遇到了一个人很不错的领导，和一群很好的小组成员。跟他们共事很开心，等我能力再提升一个档次再说吧。不知道，我还要几年可以去你的城市。我相信不会太久，我一定会出人头地的。等着我，我要给你幸福。

还是得说再见了，我收到过一个短信，‘不管以前如何，我们还是好朋友’，我知道是你发的，你也要知道，不管如何，我心里只有你一人。下次，不知道又是什么时候写信给你了，这段时间，我会尽力让自己过的充实，那么下次就有更多的话写给你了。

晚安，再会。

#给你的信
